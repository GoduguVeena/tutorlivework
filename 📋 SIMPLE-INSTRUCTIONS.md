## ?? If Something Goes Wrong:

### **"Can't connect from other devices"**
- Run the script as Administrator (right-click ? "Run as administrator")
- Make sure all devices are on same Wi-Fi network

### **"Port already in use"**
- Close any other programs using port 5000
- Try restarting your computer

### **"Database error"**
- The script handles this automatically, just try again
